# Laptop-Data-Analysis-Project_1
This project analyzes a dataset of laptops collected from an e-commerce website. It explores how features like brand, processor, RAM, storage, screen size, and graphics influence laptop prices.
Using Python in Google Colab, the project covers data preprocessing, exploratory data analysis (EDA), visualizations, insights, and interactive filters to identify patterns and trends. The goal is to understand what specifications drive laptop pricing and brand performance in the market.
